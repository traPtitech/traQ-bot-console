# BOTイベントリファレンス(タグ)

## TAG_ADDED
BOTにタグが追加された

### サンプル
```json
{
  "eventTime": "2019-05-08T08:31:06.566228282Z",
  "tagId": "2bc06cda-bdb9-4a68-8000-62f907f36a92",
  "tag": "全強"
}
```

## TAG_REMOVED
BOTからタグが削除された

### サンプル
```json
{
  "eventTime": "2019-05-08T08:31:06.566228282Z",
  "tagId": "2bc06cda-bdb9-4a68-8000-62f907f36a92",
  "tag": "全強"
}
```
